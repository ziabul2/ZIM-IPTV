# ZIM IPTV 📺

ZIM IPTV is a powerful and lightweight Android IPTV player designed for smooth live TV streaming, easy channel management, and a user-friendly experience.

## 🚀 Features

- 📺 Live IPTV streaming
- ⚡ Fast and lightweight player
- 🔄 Automatic app update checking
- 🌐 Online version control with JSON
- 🎨 Clean and simple user interface
- 📱 Android APK installation support

## 📥 Download Latest APK

Download the newest version from the GitHub Releases section.

➡️ https://github.com/ziabul2/ZIM-IPTV/releases

---

## 🔄 Update System

The app uses a `version.json` file to check:

- Current application version
- Latest available update
- Release notes
- APK download URL
- Optional force update support

## 🛠 Installation

1. Download the APK file.
2. Enable "Install Unknown Apps" on Android.
3. Install and open ZIM IPTV.
